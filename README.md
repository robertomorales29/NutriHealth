# Repositorio de versiones - NutriHealth
Este repositorio contiene dos versiones del proyecto NutriHealth, desarrollado como sistema web para la gestión de pacientes, nutriólogos y entrenadores.

## Versiones incluidas

### 1. version-demo
Contiene una versión demo preliminar del sistema. Esta versión representa el avance utilizado del proyecto antes de integrar las mejoras finales.

### 2. version-final
Contiene la versión final del sistema NutriHealth. En esta versión se integraron los últimos ajustes visuales, funcionales y de documentación necesarios para la entrega definitiva.

## Tecnologías utilizadas
- PHP
- MySQL
- XAMPP
- phpMyAdmin
- Bootstrap 5
- HTML, CSS y JavaScript
